# 截图数字求和

一个 Chrome / Edge 浏览器扩展：粘贴或上传数字截图，自动用本地 OCR 识别出图中的数字并求和。支持多张截图批量累加、逐项编辑、按图排除。

## 特点

- **完全离线运行**：OCR 引擎、识别模型全部打包在扩展本地，安装后不需要联网，也不会把截图上传到任何服务器
- **批量处理**：一次可以选择或粘贴多张截图，结果自动累加到同一个合计里
- **缩略图管理**：每张处理过的截图都会显示缩略图，点击可放大查看；每张图都能单独勾选/排除，或直接删除（连带它识别出的数字一起移除）
- **识别结果可编辑**：OCR 认错的数字可以直接改，也能手动增删单个数字
- **一键复制合计**：算好的总数一键复制到剪贴板
- **侧边栏常驻**：用 Chrome 的 Side Panel 实现，切换标签页、浏览网页都不会把已识别的结果清掉

## 安装方法（开发者模式加载，不需要发布到应用商店）

1. 下载并解压本仓库（点击右上角绿色 `Code` 按钮 → `Download ZIP`）
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 打开右上角"开发者模式"
4. 点击"加载已解压的扩展程序"，选择解压出来的文件夹
5. 点击工具栏图标，会打开侧边栏，粘贴截图（Ctrl/Cmd+V）或点击选择图片即可

## 使用说明

- 粘贴或选择图片后会自动识别，识别到的数字列在下方，逐项可编辑、可勾选/排除
- 缩略图左上角的勾选框：取消勾选会把这张图识别出的所有数字整体从合计中排除（图会变灰），图本身还保留
- 缩略图右上角的 "×"：彻底删除这张图和它对应的数字
- "+ 手动添加"：补一个 OCR 漏识别的数字
- "全选"：把所有已排除的数字/图片重新纳入合计
- "清空重来"：清空当前所有结果，重新开始
- "复制"：把合计数字复制到剪贴板

## 技术实现

- OCR 使用 [Tesseract.js](https://github.com/naptha/tesseract.js)（引擎、WASM 核心、语言包全部本地打包，不依赖任何远程接口）
- Manifest V3 扩展，使用 Side Panel API 保持面板常驻
- 纯前端实现，没有后端、没有数据收集

## 局限

- 手写体、花体字、密集小字识别率有限
- 数字之间如果用"-"分隔，可能被误判为负号（可手动编辑修正）
- 目前是把所有识别到的数字直接相加，不支持按数学表达式求值（如 `12+34-5` 会被当成 12、34、-5 三个数相加，而不是做减法运算）
